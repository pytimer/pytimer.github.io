日志设置

在使用tdlib库的时候，需要先设置日志级别

tdlib.SetLogVerbosityLevel(1)

SetLogVerbosityLevel(int)设置tdlib的日志级别，默认为5

级别 说明
0 fatal
1 error
2 warning、debug warning
3 info
4 debug
5 verbose debug
>5 and <=1024 更多的日志
